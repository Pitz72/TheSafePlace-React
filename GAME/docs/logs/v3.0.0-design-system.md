# v3.0.0 — Design System & UI Primitives

**Data:** 2026-05-15
**Tipo:** Breaking change (redesign completo)

---

## Contesto

Avvio dell'implementazione v3.0 di The Safe Place Chronicles basata sul GDD v3.0 (completato
2026-05-14). Questo step installa il design system "Diario del Sopravvissuto" che sostituisce
integralmente l'estetica CRT/terminale precedente.

## Modifiche

### Nuove dipendenze
- `@fontsource/eb-garamond` — serif narrativo (body/narrativa)
- `@fontsource/barlow-condensed` — sans condensed (UI label, bottoni, timbri)
- `@fontsource/caveat` — handwriting (annotazioni, nomi luoghi)
- Font self-hosted via @fontsource (bundle Vite, compatibile Tauri offline)

### CSS
- `src/index.css` — rimosso interamente il tema CRT (VT323, scanlines, phosphor, theme-standard/crt/high_contrast); rimpiazzato con import font + import diary.css + reset globale
- `src/styles/diary.css` — nuovo file: design language completo "Diario del Sopravvissuto"
  - Token palette cold-livid (`--tsp-paper`, `--tsp-ink`, `--tsp-rust`, `--tsp-mint`, `--tsp-slate`, ...)
  - Texture paper (`.paper`) con grana fibre, macchie acqua, noise SVG
  - Classi tipografiche: `.t-serif`, `.t-sans`, `.t-hand`, `.t-label`, `.t-stamp`
  - Primitivi fisici: `.tape`, `.pin`, `.staple`
  - Bottoni `.tsp-btn` (pressed-stamp style)
  - Meter `.tsp-meter` (hatched bar)
  - Mappa `.tsp-map` con pieghe e macchie caffè

### Componenti UI (`src/components/ui/`)
Tutti i componenti primitivi portati dal handoff a TypeScript React 19:
- `StatMeter` — barra risorse a tratteggio
- `Compass` — bussola SVG hand-drawn
- `TornEdge` — bordo strappato SVG generativo
- `Stamp` — timbro ruotato
- `Tape` — striscia scotch
- `Pin` — puntina da disegno
- `JournalEntry` — voce diario con timestamp
- `ResourceLine` — riga risorsa con etichetta/valore
- `PageHeader` — intestazione pagina (capitolo, luogo, ora, meteo)
- `PaperMap` — mappa SVG paper placeholder (verrà sostituita nella fase mappa)
- `index.ts` — barrel export

### Altri file
- `index.html` — rimosso link Google Fonts VT323; aggiornati meta description

## Compatibilità

I componenti vecchi (screens del GAME) continuano a usare le classi Tailwind. Le classi CRT
(`.theme-crt`, `.theme-standard`, ecc.) non esistono più ma i componenti che le usano non
sono ancora stati riscritti — questo è atteso. Il restyling di ogni screen avverrà negli step
successivi.
