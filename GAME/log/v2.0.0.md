# v2.0.0 - Technology Stack Refactor

**Date:** 2025-12-03
**Status:** Alpha (Refactoring Complete)

## Major Changes
The entire technology stack of "The Safe Place" has been refactored to modernize the development environment and enable future expansion.

### Core Framework
- **React 19.2.0**: Upgraded from legacy version.
- **Vite 6.2.0**: Replaced Create React App / Webpack for faster builds.
- **TypeScript ~5.8.2**: Enforced strict typing across the codebase.

### Game Engine & Logic
- **Phaser 3.90.0**: Integrated as the core game engine for map rendering and game logic (Migration in progress).
- **Zustand 5.0.8**: Replaced legacy Context API/Redux for global state management.
- **Inkjs 2.3.2**: Added for advanced branching narrative.

### Styling
- **Tailwind CSS 4.1.17**: Configured for build-time compilation, replacing CDN usage.

### Structure
- **Source Code**: All source files moved to `src/` directory following modern standards.
- **Assets**: Consolidated in `src/assets`.

## Known Issues
- Visual regressions in some UI screens due to Tailwind update.
- Map and Character currently invisible (pending full Phaser integration).
