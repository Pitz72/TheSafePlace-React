# 2.0.3 - Narrative System Migration & Retro Enhancements

## Nuove Funzionalità
- **Narrative System (Inkjs)**:
    - Migrazione completa di tutti i contenuti narrativi (Cutscenes, Dialoghi, Quest, Storia Principale) in una struttura Ink modulare.
    - Creazione di `src/assets/story/main.ink` come file master.
    - Implementazione di `NarrativeService` per gestire la logica Ink e il binding delle funzioni esterne.
    - Aggiunta di moduli specifici:
        - `modules/common.ink`: Variabili globali e funzioni helper.
        - `modules/cutscenes/`: Script delle cutscene (es. Intro).
        - `modules/dialogues/`: Dialoghi NPC (Marcus, Giona, Anya, Silas, Olivia).
        - `modules/quests/`: Logica e testi delle quest.
- **Phaser Retro Enhancements**:
    - Implementazione di un effetto CRT Scanline tramite shader personalizzato (`CRTPipeline`).
    - Aggiunta di animazioni "idle" (respiro) per il personaggio giocante.

## Modifiche Tecniche
- **Refactoring Stack Tecnologico**:
    - Aggiornamento a React 19, TypeScript ~5.8.2, Vite 6.2.0.
    - Integrazione di Phaser 3.90.0 come motore di gioco.
    - Adozione di Zustand per la gestione dello stato globale.
    - Configurazione di Tailwind CSS v3 per lo styling.
- **Struttura del Progetto**:
    - Riorganizzazione completa della cartella `src/` per una migliore modularità.
    - Centralizzazione della logica di crafting in `CraftingService`.

## Correzioni
- Risolti problemi di linting in `interactionStore.ts` e `NarrativeService.ts`.
- Ripristinata la compatibilità con Tailwind CSS v3 dopo problemi con la v4.
- Corretti i percorsi di importazione per gli asset e i componenti.

## Note
- È necessario compilare i file `.ink` in un file `story.json` e posizionarlo in `public/data/story.json` affinché il gioco carichi correttamente la narrazione.
