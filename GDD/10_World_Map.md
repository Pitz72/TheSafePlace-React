# 10. Mappa del Mondo

## 10.1 Introduzione
Questo documento contiene la rappresentazione testuale della mappa di gioco e la legenda per interpretare i simboli. La mappa è una griglia di caratteri che viene renderizzata dal motore di gioco utilizzando un tileset.

## 10.2 Legenda Simboli
Ogni carattere nella griglia della mappa corrisponde a un tipo di terreno o a un punto di interesse specifico.

| Simbolo | Descrizione | Tipo |
| :---: | :--- | :--- |
| **.** | Erba / Pianura | Terreno |
| **F** | Foresta | Terreno |
| **M** | Montagna | Terreno |
| **~** | Acqua / Fiume | Terreno |
| **C** | Città (Rovine) | Terreno |
| **#** | Muro / Ostacolo | Terreno |
| **V** | Villaggio | Terreno |
| **S** | Punto di Partenza (Start) | POI |
| **R** | Rifugio (Shelter) | POI |
| **A** | Crocevia (Crossroads) | POI |
| **H** | Erborista | POI |
| **B** | Biblioteca | POI |
| **L** | Laboratorio (Lab) | POI |
| **N** | Il Nido (Nest) | POI |
| **E** | Destinazione Finale (End) | POI |
| **T** | Commerciante (Trader) | NPC |
| **X** | Nemico (Spawn) | NPC |
