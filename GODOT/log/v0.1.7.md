# Changelog v0.1.7 - UI Overhaul

## Interface
*   **Nuovo Layout "Terminal"**:
    *   Sostituito l'HUD temporaneo con una struttura definitiva a 3 colonne.
    *   **Colonna SX**: Parametri Vitali (Health, Hunger, Thirst, Fatigue) e Controlli.
    *   **Colonna Centro**: Viewport di gioco (Mappa).
    *   **Colonna DX**: Status (Tempo, Livello, XP) e Attributi (S.P.E.C.I.A.L.).
    *   **Area Inferiore**: Log di sistema persistente.
*   **Global Theme**:
    *   Creato `terminal.theme` con palette Nero/Verde Fosfori (#33ff00).
    *   Stile "Flat" senza anti-aliasing per un look retro.
    *   Font Monospaziato applicato globalmente.

## Technical
*   **MainUI**:
    *   Nuova scena principale `MainUI.tscn` che funge da container per il `World`.
    *   Uso di `SubViewportContainer` per renderizzare il gioco all'interno dell'interfaccia.
    *   Migrazione della logica UI da `HUD.gd` a `MainUI.gd`.

## Hotfix
*   **Resource Error**:
    *   Rinominato `terminal.theme` in `terminal.tres` per risolvere un errore di parsing ("Unrecognized binary resource file"). Godot preferisce l'estensione `.tres` per le risorse testuali create manualmente.
*   **UI Tuning**:
    *   Ridimensionate le colonne laterali (20%) per dare più spazio alla Mappa (60%).
    *   Abilitato `handle_input_locally` sul SubViewport e implementato `grab_focus()` in `MainUI.gd` per garantire che i controlli da tastiera raggiungano il gioco.
    *   **Logic Fix (CRITICO)**: Risolto un bug bloccante che impediva il movimento.
        *   *Sintomo*: Il Player ignorava gli input WASD nonostante la UI fosse configurata correttamente.
        *   *Causa*: `GameManager` inizializzava lo stato di gioco su `MENU`. Lo script `Player.gd` ha un controllo di sicurezza (`if current_state != PLAYING: return`) che bloccava ogni azione.
        *   *Soluzione*: Forzato `GameManager.current_state = PLAYING` nell'inizializzazione di `MainUI.gd`.
    *   Ridotto lo zoom della Camera2D (da 2.0 a 1.0) per una visuale più ampia.
    *   **Estetica**: Aggiornato `terminal.tres` con il verde classico `#41f02d` e bordi più spessi (2px).
    *   **Font**: Ripristinato `TerminalFont` (SystemFont) poiché `PressStart2P` causava problemi di rendering (testo invisibile).
    *   **Inventario**: Aggiunto pannello Inventario funzionante nella colonna di sinistra (Tasto 'I').
