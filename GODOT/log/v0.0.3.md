# Changelog v0.0.3 - Movimento a Griglia e Camera

## Miglioramenti Gameplay

### Sistema di Movimento
*   **Grid-Based Movement**: Sostituito il movimento fisico continuo con un sistema a griglia (tile-based).
    *   Il giocatore ora si muove di esattamente 32 pixel (1 tile) alla volta.
    *   Implementato l'uso di `Tween` per rendere fluido lo spostamento tra una casella e l'altra.
    *   Il sistema previene movimenti diagonali o input multipli durante lo spostamento, garantendo un perfetto allineamento con la griglia.

### Camera System
*   **Limiti Dinamici**: La telecamera ora rispetta i confini della mappa di gioco.
    *   Aggiunta la funzione `set_camera_limits` allo script del Player.
    *   Lo script `World.gd` calcola le dimensioni totali della mappa (basandosi sui dati del generatore) e imposta i limiti della camera all'avvio.
    *   Questo impedisce al giocatore di vedere l'area "vuota" (sfondo grigio) oltre i bordi del livello.

### Fix Minori
*   **Spawn Point**: Corretto il posizionamento iniziale del giocatore per essere perfettamente centrato nella tile di partenza (offset di +16, +16 pixel).
