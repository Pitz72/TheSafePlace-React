# Changelog v0.0.8 - Sistema Inventario

## Nuove Funzionalità

### Inventario
*   **Struttura Dati**: Implementato il dizionario `inventory` nel `GameManager` per tracciare gli oggetti posseduti e le quantità.
*   **Gestione Oggetti**: Aggiunte funzioni `add_item` e `remove_item` per manipolare l'inventario.
*   **Interfaccia Diario**: La tab "Inventory" del Diario ora mostra la lista degli oggetti posseduti.
*   **Input**: Aggiunto il tasto **J** per aprire/chiudere rapidamente il Diario.

## Bug Fixes
*   **Nomi Oggetti**: Risolto un problema per cui gli oggetti di debug venivano mostrati con il loro ID interno invece del nome localizzato.
    *   Aggiornato il codice di test per utilizzare ID validi (`CONS_001`, `CONS_002`) presenti nel database JSON.
    *   Il sistema ora recupera correttamente il campo `name` dal `DataLoader`.

## Note
*   All'avvio vengono aggiunti automaticamente alcuni oggetti di prova ("Razione di cibo" e "Bottiglia d'acqua") per verificare il funzionamento della UI.
